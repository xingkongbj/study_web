## RegExp 构造函数

- 允许修改，修饰符

```
new RegExp(/abc/ig, 'i').flags
// "i"
```

## u 修饰符

- 正确处理四个字节的 UTF-16 编码

```
/^\uD83D/u.test('\uD83D\uDC2A') // false
/^\uD83D/.test('\uD83D\uDC2A') // true
```

## y 修饰符

- g修饰符只要剩余位置中存在匹配就可，而y修饰符确保匹配必须从剩余的第一个位置开始，这也就是“粘连”的涵义。

```
var s = 'aaa_aa_a';
var r1 = /a+/g;
var r2 = /a+/y;

r1.exec(s) // ["aaa"]
r2.exec(s) // ["aaa"]

r1.exec(s) // ["aa"]
r2.exec(s) // null
```

## sticky 属性

- 表示是否设置了y修饰符

## flags 属性

- 返回正则表达式的修饰符。
